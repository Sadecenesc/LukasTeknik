# Lukas Teknik

Lukas Teknik resmi web sitesi — Next.js 15, TypeScript ve Tailwind CSS ile geliştirilmiştir.

## Teknolojiler

- **Next.js 15** — App Router
- **TypeScript**
- **Tailwind CSS**
- **Supabase** — Blog ve veri yönetimi

## Kurulum

```bash
npm install
```

`.env.example` dosyasını `.env.local` olarak kopyalayın ve Supabase bilgilerinizi girin:

```bash
cp .env.example .env.local
```

## Geliştirme

```bash
npm run dev
```

Uygulama `http://localhost:3000` adresinde çalışır.

## Build

```bash
npm run build
npm start
```
